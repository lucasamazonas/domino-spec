# 🂡 DOMINO-SPEC
> Porque claramente a humanidade precisava de uma especificação formal de dominó.

Bem-vindo ao repositório que finalmente resolve o maior problema da sociedade moderna:

“Essa não é a regra certa.”

Ou seja, este projeto existe porque:
- Alguém perdeu.
- Alguém discordou.
- E alguém disse: "vamos documentar isso".

Aqui documentamos o dominó como se estivéssemos escrevendo a constituição de um pequeno país.

---

# 🎯 Propósito

Este projeto existe para:

- Padronizar regras que já eram simples.
- Evitar debates circulares de 47 minutos.
- Transformar opinião em documentação.
- Permitir que Pull Requests decidam conflitos familiares.

Se você já ouviu:
- “Sempre foi assim.”
- “Você que aprendeu errado.”
- “Aqui não vale isso.”

Este repositório é para você.

# 🔀 Como Contribuir

Quer mudar o dominó?

Ótimo. Mas faça direito.

1. Fork
2. Branch `feature/sua-ideia-controversa`
3. Arquivo em `/proposals`
4. Pull Request com argumentação minimamente racional

✅ Argumentos aceitos:
- Demonstração clara de como a regra reduz ambiguidade durante a partida;
- Evidência de que a regra diminui disputas recorrentes;
- Análise de impacto no tempo médio de jogo;
- Coerência com a mecânica estrutural do dominó (encaixe, pontuação, bloqueio);
- Consistência quando aplicada em múltiplos cenários (início, meio e final de partida);
- Compatibilidade com outras regras já formalizadas;
- Justificativa que não dependa da interpretação subjetiva de intenção;
- Teste prático documentado com resultado reproduzível;
- Proposta que não crie mais exceções do que resolve;

❌ Argumentos rejeitados

- “Mas fica mais interessante assim.” (interessante para quem?);
- “A gente sempre resolve na hora.” (justamente o problema);
- Regra que só funciona quando todo mundo já sabe o que você quis dizer;
- Proposta que exige explicação toda vez que é aplicada;
- Exceção criada para corrigir outra exceção;
- Argumento que só faz sentido quando você está ganhando;
- Regra que depende de memória seletiva;
- Mudança que aumenta a chance de discussão ao invés de reduzir;
